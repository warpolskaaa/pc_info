[RU] ОПИСАНИЕ:
Простая и быстрая утилита для получения основных характеристик 
вашего компьютера (Процессор, ОЗУ, Видеокарта, Материнская плата).

КАК ИСПОЛЬЗОВАТЬ:
1. Запустите файл .exe от имени администратора (для более точных данных).
2. Дождитесь завершения сканирования системы.
3. Программа выведет данные и будет ждать нажатия клавиши Enter для закрытия.

ТРЕБОВАНИЯ:
- ОС Windows 10/11.
- Установленные драйверы чипсета (для корректного определения частоты ОЗУ).

------------------------------------------------------------

[EN] DESCRIPTION:
A simple and fast utility to retrieve the core specifications 
of your PC (CPU, RAM, GPU, Motherboard).

HOW TO USE:
1. Run the .exe file as administrator (for more accurate data).
2. Wait for the system scan to complete.
3. The program will display the data and wait for you to press Enter to exit.

REQUIREMENTS:
- Windows 10/11 OS.
- Installed chipset drivers (for correct RAM frequency detection).
